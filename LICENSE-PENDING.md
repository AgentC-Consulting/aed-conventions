# License — pending owner decision

This repository does not yet carry a license, deliberately. Until a LICENSE
file lands, default copyright applies: **all rights reserved, AgentC
Consulting.** You may read the conventions; redistribution and derivatives
wait for the license.

## Why pending

The license choice is the attribution mechanism for this project, so it is
being decided explicitly rather than defaulted. The options under review:

1. **CC BY 4.0 for everything** — reuse and adapt with credit. Maximum spread;
   awkward for the code examples (attribution ceremony inside source files
   makes teams hesitant to paste snippets).
2. **CC BY-SA 4.0 for everything** — credit plus share-alike. Keeps
   derivatives open, but share-alike deters exactly the corporate style-guide
   adaptations that generate citations.
3. **Dual (currently favored): CC BY 4.0 for the prose, MIT for the code
   examples** — the conventions text requires attribution to AgentC
   Consulting; the example code is take-freely MIT so it can be pasted into
   any codebase, by any human or agent, with zero friction.

When the decision lands, this file is replaced by `LICENSE` (and
`LICENSE-EXAMPLES` if dual), the SPDX identifier(s) are added to
`CITATION.cff`, and the change is recorded in `ADOPTION.md` and a signed tag.
